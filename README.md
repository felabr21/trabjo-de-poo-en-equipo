# 🎟️ Sistema de Gestión de Eventos y Boletos

Proyecto integrador de Programación Orientada a Objetos desarrollado en Java.

# Descripción
Aplicación para administrar eventos y venta de boletos, aplicando herencia, 
polimorfismo, excepciones personalizadas y persistencia mediante archivos .txt.

# Tecnologías
- Java
- BlueJ
- Archivos .txt para persistencia

# Tipos de eventos
- Concierto
- Conferencia
- Teatro

# Funcionalidades
- Registrar eventos
- Vender boletos individual y múltiple
- Cancelar boletos
- Historial de ventas
- Persistencia automática de datos
